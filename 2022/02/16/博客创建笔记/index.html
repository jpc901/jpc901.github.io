博客创建笔记

发表于2022-02-16|更新于2022-02-27
|阅读量:

需要有git node hexo npm

全局安装cnpm

npm install -g cnpm –registry=https://registry.npm.taobao.org

cnpm 全局安装hexo 博客框架

cnpm install -g hexo-cli

创建blog

mkdir blog

进入blog初始化一个hexo blog

hexo init

创建文章

hexo n “我的笔记”

安装一个插件建立hexo仓库

cnpm install –save hexo-deployer-git
